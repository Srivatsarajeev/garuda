#  GarudaAI
## Smart Soldier Safety & Rescue System using AI & IoT

GarudaAI is an AI and IoT-based defense safety system designed to monitor soldiers in real-time, detect emergencies, and assist in rescue operations. The system continuously tracks soldier health conditions and location data using wearable IoT devices and intelligent monitoring algorithms.

This project combines:
- Artificial Intelligence
- Internet of Things (IoT)
- Real-time Monitoring
- FastAPI Backend
- React Frontend
- DevOps Concepts

---

#  Features

✅ Real-time Soldier Health Monitoring  
✅ Heart Rate & Temperature Tracking  
✅ GPS Location Tracking  
✅ Emergency Alert Detection  
✅ AI-based Risk Detection  
✅ Live Dashboard Monitoring  
✅ Color-coded Danger Status  
✅ IoT Device Integration (ESP32 / Simulation)  
✅ Real-time Data Visualization  

---

#  Problem Statement

Soldiers working in remote areas, forests, mountains, and war zones face high risks due to:
- Lack of real-time health monitoring
- Delayed rescue response
- Communication limitations
- Difficulty tracking injured or lost soldiers

GarudaAI aims to provide a smart monitoring and rescue support system to improve soldier safety.

---

#  Proposed Solution

GarudaAI uses IoT sensors and AI algorithms to:
1. Collect soldier health data
2. Track live location
3. Detect dangerous conditions
4. Send emergency alerts
5. Display soldier status on a dashboard

The system can work using:
- Real IoT hardware (ESP32)
- Simulated IoT data for testing/demo purposes
